Вырианты объединения

  • Создать feature

  • Создать пользовательскую историю

  • Обновить номер пользовательской истории в feature

  • Обновить путь к feature в пользовательской истории

  • Настраиваемый

    • Снала выполняетсья обновление пользовательской истории из feature файла (тех полей, для которых выбран режим объединения "Взять из agiler"), затем - feature файла из пользовательской истории (тех полей, для которых выбран режим объединения "Взять из feature"). Поля "Номер" и "Путь" игнорируются, т.к. для них есть соответствующие варианты объединения.
